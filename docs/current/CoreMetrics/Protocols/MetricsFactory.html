MetricsFactory

public protocol MetricsFactory

The MetricsFactory is the bridge between the MetricsSystem and the metrics backend implementation. MetricsFactory role is to initialize concrete implementations of the various metric types:

  • Counter -> CounterHandler
  • Recorder -> RecorderHandler
  • Timer -> TimerHandler

This type is an implementation detail and should not be used directly, unless implementing your own metrics backend. To use the SwiftMetrics API, please refer to the documentation of MetricsSystem.
